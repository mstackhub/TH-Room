จัดการห้องไลฟ์ทั้งหมด (Studio Rooms)

ID ชื่อห้อง (Room Name) ความจุ (Capacity) รายละเอียด สถานะ ดำเนินการ

จัดการแบรนด์ลูกค้า (Brand Customers)

ID ชื่อแบรนด์ (Brand Name) ผู้ดูแลแบรนด์ (ชื่อทีมงาน) สถานะ ดำเนินการ

การกำหนดสิทธิ์ผู้ใช้งาน (User Management)

ชื่อผู้ใช้งาน (Username) รหัสผ่าน (Password) ชื่อทีมงาน ระดับสิทธิ์ (Role) สถานะการใช้งาน ดำเนินการ

กำหนดสิทธิ์บทบาทและสิทธิ์พื้นฐาน (Roles Management)

ชื่อบทบาท (Role Name) คำอธิบาย สิทธิ์การจองห้อง (Create/Edit/Cancel) เป็นผู้ดูแลสูงสุด (Is Admin) แท็บที่เข้าถึงได้ (Allowed Tabs) ดำเนินการ

กรุณาใส่รหัสผ่าน Master Password

หน้าประวัติกิจกรรมเป็นข้อมูลความปลอดภัยระดับสูงสุด จำเป็นต้องกรอกรหัสผ่านเพื่อเข้าใช้งาน

บันทึกกิจกรรมความเคลื่อนไหว (Activity Audit Trail)

บันทึกการกระทำทุกประเภทเพื่อตรวจสอบย้อนหลังย้อนเวลารายการแก้ไข

วันเวลา (Date Time) ▼ ผู้กระทำ (User) กิจกรรม (Action) ค่าเริ่มต้น (Before) หลังการแก้ไข (After) IP Address อุปกรณ์ (Device)

ตั้งค่าการแจ้งเตือน LINE Messaging API

เชื่อมต่อกับระบบ LINE Official Account เพื่อส่งการแจ้งเตือนคิวงานจัดจองสตูดิโอแบบเรียลไทม์

ระบบจะจัดส่ง Flex Message สวยงามเมื่อมีการสร้าง แก้ไข หรือยกเลิกการจอง
คัดลอกจาก LINE Developers Console > Messaging API > Channel access token (long-lived)
ใส่ ID ของผู้รับ (User ID) หรือกลุ่ม (Group ID) ที่ต้องการให้ยิงแจ้งเตือนเข้าไป
ใส่ URL ของหน้าเว็บที่ผู้ใช้ทั่วไปใช้เข้าระบบ เพื่อส่งในปุ่ม LINE Flex Message
💡 วิธีการดึง Group ID หรือ User ID:
  1. เชิญ Line Bot หรือ Line OA ของคุณเข้ากลุ่ม LINE หรือ แชทส่วนตัว
  2. พิมพ์ข้อความ ขอไอดีกลุ่ม หรือ ขอไอดี ในห้องแชทนั้น
  3. บอทจะตอบกลับเป็นรหัส ID (เช่น C1a2b3... หรือ U11b2...) ให้นำค่านั้นมาวางด้านบน
